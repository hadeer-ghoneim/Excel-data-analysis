# Excel-data-analysis
Analyze real life data from the New York Stock exchange:

(1) data analysis and visualization.
(2) creating a dynamic Profit and Loss Statement dashboard.
(3) building a financial forecasting model with different scenarios.


Appling following features:

- Data Analysis: Descriptive statistics, pivot tables, and spreadsheet formulas.
- Visualization: Charts, dashboards, and summary reports.
- Financial Modeling: Profit & Loss statement, forecasting (Best, Base, Weak scenarios).
- Data Cleaning: Organizing and preparing structured data.
- Presentation: Slide deck summarizing insights and findings.
